| junit4       | junit5      | 特点                                                     |
| ------------ | ----------- | -------------------------------------------------------- |
| @BeforeClass | @BeforeAll  | 在当前类的所有测试方法之前执行。注解在【静态方法】上。   |
| @AfterClass  | @AfterAll   | 在当前类中的所有测试方法之后执行。注解在【静态方法】上。 |
| @Before      | @BeforeEach | 在每个测试方法之前执行。注解在【非静态方法】上。         |
| @After       | @AfterEach  | 在每个测试方法之后执行。注解在【非静态方法】上           |
